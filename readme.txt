The HTML file implements basic fuctionalloity of configuring a network device.
The page has been developed basic HTML, jquery and javascript and css. Have included the css declarations has been included in the same file but typically should be included as a separate file.
On right clicking the image of a switch a menu is displayed listing the  various configuration options. On clicking on any one of the options a dialog box is displays seeking input from the user. Basic input validation for IP addresses has been implemented. On clicking the applied button an alert is displayed signifying that the changes have been applied. On clicking the cancel button the dialog box cease to be displayed.
